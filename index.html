🔗 구글 캘린더 실제 연동 방법
⚠️ 현재는 데모 모드예요. 실제 연동을 위해 아래 3단계가 필요해요.
① Google Cloud Console 설정
console.cloud.google.com → 새 프로젝트 → Google Calendar API 활성화 → OAuth 2.0 클라이언트 ID 발급 → 승인된 도메인에 Vercel URL 추가
② index.html 상단에 Client ID 입력
const GCAL_CLIENT_ID = "xxx.apps.googleusercontent.com";
const GCAL_API_KEY = "AIza...";
③ 아내 캘린더 공유
구글 캘린더에서 아내 캘린더를 내 계정과 공유하면 두 사람 일정이 동시에 표시돼요.
💡 현재 데모 모드에서도 캘린더 UI, 코디 계획, 세탁 트래커, AI 추천은 모두 실제 작동해요.
